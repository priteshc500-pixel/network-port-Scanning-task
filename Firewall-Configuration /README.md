# Task 4 - Firewall Configuration

## Objective
Configure and test basic firewall rules.

## Tool Used
Windows Defender Firewall

## Steps Performed

1. Opened Windows Firewall.
2. Viewed inbound rules.
3. Created a rule blocking TCP port 23.
4. Verified the rule.
5. Removed the rule after testing.

## Why Port 23?

Port 23 is used by Telnet, which transmits data in plain text and is considered insecure.

## Conclusion

This task helped me understand firewall configuration, inbound rules, and network traffic filtering.
